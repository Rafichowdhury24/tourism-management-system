# Tourism Management System

## Overview
The Tourism Management System is a database-based web application developed to manage tourism services such as destinations, bookings, hotels, transportation, payments, and tour packages.

---
## Useful Links

- [GitHub Repository](https://github.com/Rafichowdhury24/tourism-management-system)
- [ERD](https://drive.google.com/file/d/1SfV62WkOu_8MGpXkGvPpK3RYLQGfjKZh/view?usp=sharing)
---

## Features
- User registration and login
- Browse destinations and tour packages
- Hotel and transportation management
- Booking and payment system
- Admin activity tracking

---

- HTML
- CSS

- Git & GitHub
- Visual Studio Code

## Database Tables
- users
- destinations
- hotels
- transportation
- tour_packages
- bookings
- payments
- admin_activity_log

---

## How to Run
1. Clone the repository
2. Import the SQL database files
3. Open the project in VS Code
4. Run `index.html` using Live Server
